# Cyclistic Bike-Share Case Study

### Overview
This case study showcases the full data analysis process carried out in a fictional business environment. The goal was to identify how different types of riders use Cyclistic bikes differently in order to deliver insights and recommendations for ways the marketing team can campaign for converting casual riders into annual members. 

The full report is hosted on RPubs. To view, click: 

[Complete Analysis Report](https://rpubs.com/tobygriffin/cyclistic_case_study)
